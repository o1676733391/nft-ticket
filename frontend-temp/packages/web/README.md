# Web App - Next.js 14

NFT Ticketing System web application built with Next.js 14.

## Setup

```bash
npm install
```

## Development

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000)

## Build

```bash
npm run build
npm run start
```
