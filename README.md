# Ticketbox Expo (scaffold)

This is a minimal Expo + React Native scaffold for web and mobile to help you start building the UI shown in the attachment.

## Quick start (PowerShell on Windows)

1. Install dependencies:

```powershell
cd d:/THU3/ticketbox-expo
npm install
```

2. Start the dev server (web + mobile):

```powershell
npm run web
# or
npm start
```

3. For mobile: install Expo Go on your phone, open the QR code shown by the dev server.

## Notes
- This scaffold contains a simple `App.js` and `src/HomeScreen.js` as a starting point.
- You can replace the placeholder images/components with your real assets and styles that match the provided web screenshot.
