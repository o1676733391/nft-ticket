# Mobile App - React Native with Expo

NFT Ticketing System mobile application.

## Setup

```bash
npm install
```

## Development

```bash
# Start development server
npm run dev

# Run on Android
npm run android

# Run on iOS
npm run ios
```

## Build

```bash
# Build for Android
npm run build:android

# Build for iOS
npm run build:ios
```
